  AI-powered tool to find companies, get contact info, and send outreach emails.
  
  Pipeline Steps
  1. Search — AI discovers companies by type/region
  2. Review — Select companies to enrich
  3. Enrich — Fetch contacts (Apollo → Hunter → Cache)
  4. Dashboard — Push to CRM / export CSV
  5. Email — Send outreach emails
  6. Complete — Summary & start new search
